Command Block¶

Basics

  • @a : all players on the server.
  • @p : nearest player
  • @r : random player

give <player> <object> <amount> <datavalue>

clear <player> <object> <datavalue>

difficulty

  • 0 : peaceful
  • 1 : easy
  • 2 : nornal
  • 3 : hard

gademode <mode> <player>

say <msg> tell <player> <msg>

spawnpoint <player> <X> <Y> <Z>

time set <value> time add <value>

weather

  • clear
  • rain
  • thunder

xp <+/-value> <player>

filtering

[r=<value>]

  • r : radius
  • rm : radius min
  • l : level
  • lm : level min
  • m : game mode
  • c : count
  • x
  • y
  • z
  • r
